# NotTooLate

**NotTooLate** is an AI-assisted, biology-aware strength training platform designed for adults aged 30+.

## Why NotTooLate?
From the age of 30, muscle mass, bone density, and functional strength decline progressively.
Cardio alone is insufficient to prevent this decline.

NotTooLate focuses on:
- Strength preservation
- Injury prevention
- Explainable exercise guidance
- Age- and sex-aware personalization

## Features
- Personalized strength routines based on age, time, activity level, and equipment
- Explainable exercise recommendations
- Beginner-safe progression
- Dark / Light theme support
- Scalable architecture for mass wellness programs

## Tech Stack
- Frontend: HTML, CSS
- Backend: Flask (Python)
- AI: LLM-based plan generation
- Styling: CSS variables with theme support

## Future Scope
- Integration with registered gyms and certified trainers
- Online coaching sessions
- Nutrition guidance partnerships
- Long-term progress tracking
